# AI for Brainstorming & Ideation

## Lesson 4: Creative Problem-Solving

### Overview
When facing challenges, AI helps you find solutions you wouldn't think of alone.

---

### Problem-Solving Techniques

**Define the problem clearly**
- What's really the core issue?

**Reframe the problem**
- State it in different ways
- New framings reveal new solutions

**Ask why repeatedly**
- Get to root causes

**Look for analogies**
- How have similar problems been solved elsewhere?

**Challenge assumptions**
- What are you assuming?
- What if those assumptions are wrong?

**Combine solutions**
- Blend ideas from different approaches

**Think about prevention**
- Can you avoid the problem entirely?

**Consider doing nothing**
- What happens if you don't solve this?
